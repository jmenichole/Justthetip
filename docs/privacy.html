3. Data Security

  • We implement security measures to protect your data, including encrypted database connections.

4. Data Retention

  • We retain user data only as long as necessary for bot functionality.

5. Third-Party Services

6. Changes to Privacy Policy

We may update this policy periodically. Continued use of the bot after changes constitutes acceptance of the updated policy.

7. Contact
